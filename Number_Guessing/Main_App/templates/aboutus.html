About Us

The Number Guessing Game is a fun and interactive game where the player tries to guess a randomly generated number within a specified range. It is a simple yet engaging game that tests the player's logic and intuition.

Features:

1. Random Number Generation: The program generates a random number between 0 and 999 for the player to guess.
2. Multiple Attempts: The player is given a certain number of attempts to guess the correct number.
3. Hint System: If the player's guess is incorrect, the program provides hints to guide the player. It indicates whether the actual number is greater or smaller than the guessed number.
4. Interactive Interface: The program has an attractive and user-friendly interface to enhance the gaming experience.
5. Winning Condition: When the player guesses the correct number, the program displays a message indicating that the player has won.
6. Losing Condition: If the player runs out of attempts without guessing the correct number, the program displays a message indicating that the player has lost and reveals the actual number.
7. Replay Option: After each game, the program gives the player the choice to play again. If selected, a new random number is generated, and the game restarts.
8. Range Limit: The numbers to be guessed are limited to the range of 0 to 999, ensuring a challenging yet manageable gameplay experience.

How to Play:

1. The computer will generate a random number between 0 and 999. This number will be the target number that the player needs to guess.
2. The player will be prompted to enter their guess.
3. The player enters a number as their guess.
4. The computer will check if the player's guess is equal to the target number.
5. If the guess is correct, the player wins the game and receives a congratulatory message. The game ends.
6. If the guess is incorrect, the computer will provide a hint to the player about whether the target number is greater or smaller than their guess.
7. The player will have a few more chances to guess the correct number. The number of chances can be predefined or based on your preference.
8. Steps 3 to 7 will be repeated until one of the following conditions is met: a. The player guesses the correct number. b. The player runs out of chances without guessing the correct number.
9. If the player runs out of chances without guessing the correct number, the computer will reveal the target number and display a message indicating that the player has lost the game.
10. After the game ends, the player can choose to play again or exit the game.
